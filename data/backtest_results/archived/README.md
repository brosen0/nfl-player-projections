# Archived Backtest Results

These results were archived on 2026-04-09 because they are unreliable.

## ts_backtest_2024_20260213_235745

**R² = 0.996** — implausibly perfect for NFL fantasy point prediction (realistic
ceiling is ~0.2-0.4 out-of-sample). This was generated by an older version of
`TimeSeriesBacktester.leakage_safe_features()` that computed train features from
a combined train+test block, allowing test data to contaminate train-period
rolling statistics. The function has been fixed to compute train features from
train data only.

Do not use these results as a baseline for comparison.
